# 🏅 Olympic Data Analysis

An interactive data analytics dashboard built using **Python**, **Streamlit**, **Pandas**, and **Matplotlib** to explore and visualize historical Olympic Games data. The application enables users to analyze medal distributions, athlete performances, country-wise achievements, and Olympic trends through interactive charts and filters.

---

## 📌 Problem Statement

Analyzing historical Olympic datasets manually is challenging due to the large volume of data and the lack of intuitive visualization tools. Extracting meaningful insights such as medal trends, top-performing countries, athlete achievements, and sport-wise statistics requires significant effort.

---

## 💡 Solution

Developed a web-based interactive dashboard that transforms raw Olympic datasets into meaningful visual insights. Users can filter data by year, country, sport, and athlete to perform exploratory data analysis and understand Olympic performance trends efficiently.

---

## ✨ Features

- 🥇 Country-wise medal tally analysis
- 🌍 Year-wise Olympic statistics
- 🏃 Athlete performance analysis
- 🏅 Sport-wise medal distribution

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/Olympic-Data-Analysis.git
cd Olympic-Data-Analysis
```

### 2. Create a Virtual Environment (Optional)

```bash
python -m venv venv
```

Activate the environment:

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
streamlit run app.py
```

The application will open in your browser.
